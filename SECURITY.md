# Security Policy

## Supported Versions

Currently, the `main` branch is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| `main`  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to the repository maintainer or open a private security advisory on GitHub. All security vulnerabilities will be promptly addressed.

Please do not report security vulnerabilities through public GitHub issues.

When reporting a vulnerability, please provide:
* A description of the vulnerability.
* Steps to reproduce the issue.
* Potential impact.

We will try our best to address the issues quickly.
